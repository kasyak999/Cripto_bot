# Скрипт для биржи
Описание скрипта

### Установка
1. Создать виртуальное окружение
    ```
    python -m venv venv
    source venv/bin/activate
    ```
2. Установить зависимости
    ```
    pip install -r requirements.txt
    ```
3. Заупуск
    ```
    python main.py 
    ```
### Файл .env
```
API_KEY=публичный_ключ
API_SECRET=секретный_ключ
DEMO=True - демо режим / False - не демо
```
